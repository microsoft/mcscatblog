# Copilot Studio Resources

Your guide to Microsoft Copilot Studio documentation, training, tools, and best practices.

## Copilot Studio Documentation (Start here!)

- [What is Copilot Studio?](https://learn.microsoft.com/en-us/microsoft-copilot-studio/fundamentals-what-is-copilot-studio) - Introduction to Microsoft Copilot Studio
- [Microsoft Copilot Studio Guidance Documentation](https://learn.microsoft.com/en-us/microsoft-copilot-studio/guidance/) - Best practices, implementation, and architecture guidance information from the team that works with our enterprise customers
- [Copilot Studio Implementation Guide - PPT Download](https://aka.ms/CopilotStudioImplementationGuide) - Microsoft CAT's end-to-end blueprint for planning, architecting, governing, and deploying Copilot Studio at scale
- [Configure high-quality instructions for generative orchestration](https://learn.microsoft.com/en-us/microsoft-copilot-studio/guidance/generative-orchestration) - Guide to configuring high-quality instructions
- [Copilot Studio Prompting Guide](https://learn.microsoft.com/en-us/microsoft-copilot-studio/create-custom-prompt) - Comprehensive guide to prompt engineering and best practices for Copilot Studio
- [Architectural Center](https://learn.microsoft.com/en-us/power-platform/architecture/) - Microsoft Power Platform and Copilot Studio Architecture Center containing guidance and reference architectures. 
- [Governance Guidance](https://learn.microsoft.com/en-us/microsoft-copilot-studio/security-and-governance) - Security and governance for Microsoft Copilot Studio
- [How to Evaluate Agents](https://www.microsoft.com/en-us/microsoft-copilot/blog/copilot-studio/how-to-evaluate-ai-agents/?msockid=3095307a1e626e5c278426931f156f05) & [Evaluation Setup](https://learn.microsoft.com/en-us/microsoft-copilot-studio/analytics-agent-evaluation-create) - About agent evaluation in Microsoft Copilot Studio

## Training and Workshops

### Self-Paced Hands-On Training

- [Copilot Studio Agent Academy](https://microsoft.github.io/agent-academy/) - Hands-on skill-building path for Copilot Studio makers, from fundamentals to advanced agent design
- [Microsoft Copilot Studio Labs](https://microsoft.github.io/mcs-labs/) - Hands-on labs for building AI agents with Microsoft Copilot Studio across multiple learning paths
- [Copilot Studio Developer Camp](https://microsoft.github.io/copilot-camp/pages/custom-engine/) - Additional Copilot Studio Custom Agent Labs. Broader Developer Camp includes paths for Copilot and Azure Foundry. 
- [Microsoft Applied Skills Certification](https://learn.microsoft.com/en-us/credentials/applied-skills/create-agents-in-microsoft-copilot-studio/) - Create agents in Microsoft Copilot Studio

### Microsoft-Led Trainings and Workshops

- [CAT AI Webinars](https://microsoft.github.io/powercat/programs/ai-webinar.html) - Microsoft CAT AI Webinars are a series of interactive virtual sessions designed to help your organization adopt Microsoft Copilot and agents
- [CAT Architecture Boot Camps](https://microsoft.github.io/powercat/programs/architecture-bootcamp.html) - Intensive hands-on workshops for designing and building enterprise-scale Copilot solutions
- [CAT Copilot Studio Kickstarter Workshops](https://kickstarter.powerappsportals.com/Available-Events/?workshopid=CS) - Workshops to help organizations define their AI vision, identify high-value agents, and scale secure, enterprise-grade AI with confidence
- [Power Up Program](https://powerup.microsoft.com/) - Microsoft's free cohort-based learning program combining structured courses with peer collaboration for Power Platform, Copilot Studio, and AI fundamentals

### Must-Follow YouTube Channels

- [The Copilot Studio Dude](https://www.youtube.com/@Dewain27) - YouTube series managed by Dewain Robinson from the Microsoft CAT team featuring an excellent 101 learning series, specific use case tutorials, advanced patterns, and best practices
- [DamoBird365](https://www.youtube.com/@damobird365) - Community-driven YouTube channel featuring Power Platform and Copilot Studio tutorials, tips, and real-world implementation examples
